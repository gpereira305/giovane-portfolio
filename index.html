eCommerce Outfit Store

Javascript Node JS Mongo DB Heroku

Esse projeto foi desenvolvido utilizando features do JavascriptES6, com orientação à objetos, API desenvolvida em Node e Express.JS, conectado ao MongoDB. O objetivo principal é simular um carrinho de compras, com seleção do item, passando pelo processo cadastro de dados para poder fazer o checkout. Possui painel admin com histórico de compras e dados dos clientes com datas de compras e status. Muitos detalhes desse projeto foram implementados por mim com o que eu aprendi durante o processo de construção. Deploy feito na plataforma Heroku.

eCommerce Tech Shop

React JS Redux Node JS Mongo DB Heroku

Esse é um dos meus projetos mais avançados até então desenvolvido utilizando M.E.R.N Stack (MongoDB, Express, React, e Node), com o objetivo de simular um mini-ecommerce, com cadastro, edição e exclusão de produtos com painel administrativo completo.
